## 安装 Mod

### 从 ZIP 文件安装

1. 进入实例设置 → **「Mod 管理」** 选项卡
2. 点击 **「从 ZIP 安装」** 按钮
3. 选择 ZIP 文件
4. 等待安装完成

### 从文件夹安装

1. 进入实例设置 → **「Mod 管理」** 选项卡
2. 点击 **「从文件夹安装」** 按钮
3. 选择包含 Mod 的文件夹
4. 等待安装完成

### 从 NexusMods 安装

1. 进入实例设置 → **「自动安装」** 选项卡
2. 搜索想要的 Mod
3. 点击 **「下载」** 或 **「安装」**
4. 等待下载和安装完成

### 拖放安装

直接将 ZIP 文件拖放到启动器窗口，即可快速安装。

## Mod 列表

Mod 列表显示所有已安装的 Mod，包含以下信息：

| 列 | 说明 |
|----|------|
| **状态** | 启用/禁用状态 |
| **名称** | Mod 显示名称 |
| **版本** | Mod 版本号 |
| **作者** | Mod 作者 |
| **ID** | Mod 唯一标识符 |

### 筛选和排序

- **搜索框** - 按名称搜索 Mod
- **状态筛选** - 仅显示启用/禁用的 Mod
- **排序** - 按名称、版本、作者排序

## 启用/禁用 Mod

### 单个 Mod

- 点击 Mod 左侧的复选框
- 或右键点击 → **「启用」** / **「禁用」**

### 批量操作

1. 按住 `Ctrl` 点击多个 Mod
2. 右键点击 → **「批量启用」** / **「批量禁用」**

## Mod 详情

点击 Mod 名称，可以查看详细信息：

- **描述** - Mod 功能描述
- **版本信息** - 支持的游戏版本
- **依赖关系** - 必需和可选依赖
- **兼容性** - 与其他 Mod 的兼容情况

## 依赖管理

SVL 会自动检测 Mod 的依赖关系：

### 依赖类型

| 类型 | 说明 |
|------|------|
| **必需依赖** | Mod 运行必须安装的依赖 |
| **可选依赖** | 可选安装，提供额外功能 |

### 自动解析

当启用一个 Mod 时，SVL 会：

1. 检查所有必需依赖是否已安装
2. 如果缺少依赖，显示警告
3. 提供一键安装缺失依赖的选项

### 拓扑排序

SVL 使用拓扑排序算法，确保 Mod 按正确的顺序加载。

## 冲突检测

SVL 会自动检测以下冲突：

### ID 冲突

- 多个 Mod 使用相同的唯一 ID
- 可能导致 Mod 加载失败

### 文件冲突

- 多个 Mod 修改同一游戏文件
- 可能导致游戏崩溃或行为异常

### 依赖冲突

- Mod 要求不兼容的依赖版本
- 可能导致运行时错误

### 查看冲突

1. 点击 **「检查冲突」** 按钮
2. 查看冲突报告
3. 根据建议解决冲突

## Mod 卸载

### 卸载单个 Mod

1. 右键点击 Mod
2. 选择 **「卸载」**
3. 确认卸载

### 批量卸载

1. 选中多个 Mod
2. 右键点击 → **「批量卸载」**

> ⚠️ **注意**：卸载 Mod 前请确保没有其他 Mod 依赖它。
