# 下载管理器

SVL 内置多任务下载管理器，支持断点续传和队列管理。

## 打开下载管理器

- 点击底部状态栏的 **「下载」** 图标
- 或使用快捷键 `Ctrl + D`

## 下载任务

### 任务类型

| 类型 | 说明 |
|------|------|
| **Mod** | 从 NexusMods / CurseForge 下载的 Mod |
| **SMAPI** | SMAPI 安装包 |
| **Modpack** | 整合包文件 |
| **更新** | 启动器或 Mod 更新 |

### 任务信息

每个任务显示：

- **名称** - 文件名称
- **大小** - 文件大小
- **进度** - 下载进度条
- **速度** - 当前下载速度
- **状态** - 任务状态

## 任务状态

| 状态 | 说明 |
|------|------|
| ⏳ **等待中** | 在队列中等待 |
| ⬇️ **下载中** | 正在下载 |
| ⏸️ **已暂停** | 用户暂停 |
| ✅ **已完成** | 下载完成 |
| ❌ **失败** | 下载失败 |
| 🔄 **重试中** | 正在重试 |

### 任务操作

- **暂停** - 暂停当前下载
- **继续** - 继续暂停的下载
- **取消** - 取消下载
- **重试** - 重试失败的下载
- **打开文件夹** - 打开下载目录

## 断点续传

SVL 支持断点续传功能：

### 自动续传

- 下载中断后，自动从断点继续
- 关闭启动器后，进度会被保存
- 重新打开后自动继续下载

### 手动续传

1. 找到暂停的任务
2. 点击 **「继续」** 按钮
3. 下载会从断点继续

## 下载设置

### 并发设置

进入 **「设置」** → **「下载」**：

| 设置 | 说明 |
|------|------|
| **并发任务数** | 同时下载的任务数量（1-5） |
| **下载目录** | 下载文件的保存位置 |
| **自动安装** | 下载完成后自动安装 |

### 网络设置

| 设置 | 说明 |
|------|------|
| **代理设置** | 配置 HTTP/SOCKS 代理 |
| **超时时间** | 连接超时时间 |
| **重试次数** | 失败后的重试次数 |

### 下载来源

SVL 支持从多个来源下载：

- **NexusMods** - 主要 Mod 平台
- **CurseForge** - 备用 Mod 平台
- **GitHub** - 开源 Mod 和更新
- **直链** - 自定义下载链接
