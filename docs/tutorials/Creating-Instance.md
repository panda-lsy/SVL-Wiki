## 什么是实例？

实例是一个独立的游戏环境，拥有：

- **独立的 Mod 列表** - 每个实例可以安装不同的 Mod
- **独立的存档** - 存档不会互相干扰
- **独立的配置** - 游戏设置可以不同

### 使用场景

| 场景 | 说明 |
|------|------|
| **多 Mod 配置** | 一个实例玩农场 Mod，另一个玩冒险 Mod |
| **版本测试** | 测试新 Mod 不影响主力配置 |
| **整合包管理** | 每个整合包使用独立实例 |

## 创建新实例

### 从主页创建

1. 点击主页左侧的 **「+」** 按钮
2. 选择 **「创建新实例」**

### 填写信息

| 字段 | 必填 | 说明 |
|------|------|------|
| **名称** | ✅ | 实例显示名称，如「我的农场」 |
| **描述** | ⚪ | 实例描述，方便识别 |
| **图标** | ⚪ | 自定义实例图标 |
| **游戏路径** | ✅ | 选择游戏基础路径 |

### 高级选项

展开 **「高级选项」** 可以设置：

| 选项 | 说明 |
|------|------|
| **版本隔离** | 启用后实例数据完全独立 |
| **自动安装 SMAPI** | 创建实例后自动安装 SMAPI |
| **复制 Mod 列表** | 从现有实例复制 Mod 配置 |

### 完成创建

1. 点击 **「创建」**
2. 等待 SVL 创建实例目录
3. 实例会出现在主页列表中

## 实例设置

进入实例后，可以配置以下选项：

### 概览

- 查看实例基本信息
- 查看 Mod 数量和状态
- 查看 SMAPI 版本
- 启动游戏

### 自动安装

- 搜索 NexusMods Mod
- 一键下载和安装
- 查看热门 Mod

### 设置

| 设置 | 说明 |
|------|------|
| **名称** | 修改实例名称 |
| **描述** | 修改实例描述 |
| **图标** | 更换实例图标 |
| **版本隔离** | 启用/禁用版本隔离 |
| **启动参数** | 自定义游戏启动参数 |

### Mod 管理

- 查看所有已安装 Mod
- 启用/禁用 Mod
- 查看 Mod 详情
- 检查依赖和冲突

### 导出

- 导出为整合包
- 分享给其他玩家

## 克隆实例

克隆可以复制现有实例的所有配置和 Mod。

### 步骤

1. 右键点击实例卡片
2. 选择 **「克隆实例」**
3. 输入新实例名称
4. 选择克隆选项：

| 选项 | 说明 |
|------|------|
| **复制 Mod 文件** | 复制所有 Mod 文件 |
| **复制配置** | 复制 Mod 配置文件 |
| **复制存档** | 复制游戏存档 |

5. 点击 **「克隆」**
6. 等待完成

### 克隆用途

- 备份当前配置
- 测试新 Mod
- 创建 Mod 配置变体

## 删除实例

### 步骤

1. 右键点击实例卡片
2. 选择 **「删除实例」**
3. 确认删除

### 注意事项

> ⚠️ **警告**：删除实例会删除：
> - 实例配置
> - Mods 文件夹
> - 隔离的存档数据

> ✅ 原版游戏文件不会被删除

### 保留数据

如果只想从列表移除但保留文件：

1. 进入实例设置 → **「设置」**
2. 点击 **「从列表移除」**
3. 选择是否保留文件
