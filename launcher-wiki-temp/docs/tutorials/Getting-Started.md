## 设置游戏路径

### 自动检测

SVL 会自动检测以下路径：

- **Steam**: `Steam\steamapps\common\Stardew Valley`
- **GOG**: `GOG Galaxy\Games\Stardew Valley`

### 手动设置

如果自动检测失败：

1. 进入 **「设置」** → **「通用」**
2. 点击 **「游戏基础路径」** 旁边的 **「浏览」**
3. 选择 Stardew Valley 的安装目录
4. 点击 **「确定」**

### 验证路径

正确的游戏目录应包含：

```
Stardew Valley/
├── Stardew Valley.exe
├── Stardew Valley.dll
├── Content/
└── ...
```

## 创建第一个实例

### 步骤

1. 点击主页左侧的 **「+」** 按钮
2. 选择 **「创建新实例」**
3. 填写信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| **名称** | 实例显示名称 | 「我的 Mod 实例」 |
| **描述** | 实例描述（可选） | 「常用的 Mod 配置」 |
| **游戏路径** | 基础游戏路径 | 自动填充 |

4. 启用 **「版本隔离」**（推荐）
5. 点击 **「创建」**

### 等待创建

SVL 会自动：
- 创建实例目录
- 设置 Mods 文件夹
- 配置隔离环境

## 安装 SMAPI

### 为什么需要 SMAPI？

SMAPI 是星露谷物语的 Mod 加载器，大多数 Mod 都需要它才能运行。

### 安装步骤

1. 选择刚创建的实例
2. 在实例概览页面，点击 **「安装 SMAPI」**
3. 选择 SMAPI 版本：
   - 推荐：**最新稳定版**
4. 点击 **「安装」**
5. 等待下载和安装完成

### 验证安装

安装完成后，你会看到：
- SMAPI 版本号
- ✓ 已安装标记

## 安装第一个 Mod

### 方法一：从 NexusMods 安装

1. 进入实例设置 → **「自动安装」** 选项卡
2. 搜索想要的 Mod（如 "NPC Map Locations"）
3. 点击搜索结果
4. 选择版本
5. 点击 **「下载并安装」**

### 方法二：从 ZIP 文件安装

1. 下载 Mod 的 ZIP 文件
2. 将 ZIP 文件拖放到 SVL 窗口
3. 或者在 **「Mod 管理」** 中点击 **「从 ZIP 安装」**

### 启用 Mod

1. 进入 **「Mod 管理」** 选项卡
2. 确保 Mod 左侧的复选框已勾选
3. 查看是否有依赖提示

## 启动游戏

### 启动方式

1. 在主页找到你的实例卡片
2. 点击 **「启动」** 按钮
3. 或双击实例卡片

### 启动模式

| 模式 | 说明 |
|------|------|
| **Mod 模式** | 通过 SMAPI 启动，加载所有启用的 Mod |
| **原版模式** | 直接启动游戏，不加载 Mod |

### 首次启动

首次启动时，SMAPI 会：
- 显示控制台窗口
- 加载所有 Mod
- 输出加载日志

如果一切正常，游戏会正常启动并显示 SMAPI 版本信息。
