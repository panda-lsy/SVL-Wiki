## 登录 NexusMods

### OAuth 登录

1. 进入 **「设置」** → **「NexusMods」**
2. 点击 **「登录」** 按钮
3. 在浏览器中完成授权
4. 返回启动器，登录完成

### 登录状态

登录后，你可以：

- 获取 Premium 下载速度
- 访问成人内容 Mod
- 使用 Collection 功能
- 追踪下载历史

## 搜索 Mod

### 基本搜索

1. 进入实例设置 → **「自动安装」** 选项卡
2. 在搜索框输入关键词
3. 按 `Enter` 或点击搜索按钮
4. 浏览搜索结果

### 高级筛选

| 筛选器 | 说明 |
|--------|------|
| **类别** | 按 Mod 类别筛选 |
| **时间** | 按发布/更新时间筛选 |
| **排序** | 按下载量、评分、时间排序 |

## 浏览 Mod 详情

点击搜索结果中的 Mod，可以查看：

- **名称和描述** - Mod 基本信息
- **截图和预览** - Mod 效果展示
- **版本列表** - 所有可用版本
- **依赖要求** - 必需的依赖 Mod
- **兼容性信息** - 支持的游戏版本

### 版本选择

在版本列表中，你可以：

- 查看每个版本支持的 SMAPI 版本
- 查看更新日志
- 选择特定版本下载

## 下载 Mod

### 下载到当前实例

1. 在 Mod 详情页选择版本
2. 点击 **「下载并安装」**
3. 等待下载和安装完成

### 仅下载

1. 点击 **「仅下载」**
2. Mod 会保存到下载目录
3. 稍后可以手动安装

### 下载选项

| 选项 | 说明 |
|------|------|
| **自动安装** | 下载后自动安装到当前实例 |
| **仅下载** | 仅下载文件，不自动安装 |
| **另存为** | 选择保存位置 |

## NXM 协议支持

SVL 支持 `nxm://` 协议，可以从浏览器一键安装 Mod。

### 启用协议关联

首次使用时，SVL 会自动注册 NXM 协议。

也可以手动启用：
1. 进入 **「设置」** → **「NexusMods」**
2. 启用 **「NXM 协议关联」**

### 使用方法

1. 在 NexusMods 网站点击 **「Download with Manager」**
2. 浏览器会打开 SVL
3. SVL 自动开始下载

## Collection 支持

SVL 支持 NexusMods Collection（整合包）功能。

### 安装 Collection

1. 在 NexusMods 网站浏览 Collection
2. 点击 **「Add to SVL」**
3. SVL 会自动打开 Collection 安装向导
4. 选择目标实例
5. 等待所有 Mod 下载安装完成

### Premium 用户

Premium 用户可以：
- 一键安装整个 Collection
- 跳过手动确认步骤
- 更快的下载速度

### 非Premium 用户

非 Premium 用户可以使用向导模式：
- 按阶段分批安装
- 手动确认每个 Mod
- 需要更多操作步骤

## API 限制

NexusMods API 有请求频率限制。

### 限制规则

| 类型 | 限制 |
|------|------|
| **每小时** | 300 次请求 |
| **每天** | 3000 次请求 |

### 查看配额

1. 进入 **「设置」** → **「NexusMods」**
2. 查看 **「API 配额」** 部分

### 配额优化

SVL 会自动缓存 API 响应，减少不必要的请求。
