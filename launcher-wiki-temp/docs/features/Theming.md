# 主题与外观

SVL 支持自定义主题和外观，让你可以个性化启动器的视觉效果。

## 主题切换

### 预设主题

SVL 提供以下预设主题：

| 主题 | 说明 |
|------|------|
| **浅色** | 明亮的浅色主题 |
| **深色** | 护眼的深色主题 |
| **跟随系统** | 自动跟随 Windows 系统主题 |

### 切换主题

1. 进入 **「设置」** → **「外观」**
2. 选择 **「主题」** 选项
3. 选择想要的主题

## 自定义配色

### 强调色

你可以自定义启动器的强调色：

1. 进入 **「设置」** → **「外观」**
2. 点击 **「强调色」** 选择器
3. 选择或输入颜色值

### 自定义主题

高级用户可以创建自定义主题：

1. 进入 **「设置」** → **「外观」**
2. 点击 **「高级设置」**
3. 修改各项颜色值：

| 颜色项 | 说明 |
|--------|------|
| **背景色** | 主背景颜色 |
| **前景色** | 主要文字颜色 |
| **强调色** | 按钮和链接颜色 |
| **边框色** | 边框和分割线颜色 |
| **卡片背景** | 卡片组件背景色 |

## 界面布局

### 侧边栏

- 可以隐藏/显示侧边栏
- 调整侧边栏宽度
- 自定义侧边栏图标

### 卡片视图

- 调整实例卡片大小
- 选择列表/网格视图
- 自定义卡片显示信息

### 首页布局

- 自定义首页模块顺序
- 显示/隐藏特定模块

## 动画效果

### 窗口动画

| 设置 | 说明 |
|------|------|
| **启用动画** | 开启/关闭界面动画 |
| **动画速度** | 调整动画播放速度 |

### 通知动画

- 弹出通知的动画效果
- 悬浮提示的显示效果
