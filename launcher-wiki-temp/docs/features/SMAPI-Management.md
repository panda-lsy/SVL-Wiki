SMAPI (Stardew Modding API) 是星露谷物语的核心 Mod 加载器。SVL 提供完整的 SMAPI 管理功能。

## 什么是 SMAPI？

SMAPI 是星露谷物语最流行的 Mod 加载器，提供：

- Mod 加载和管理
- Mod 依赖解析
- 错误日志和调试
- Mod 更新检测
- 兼容性检查

### 为什么需要 SMAPI？

大多数星露谷物语 Mod 都需要 SMAPI 才能运行。SVL 的大多数功能也依赖于 SMAPI。

## 安装 SMAPI

### 自动安装

1. 选择一个游戏实例
2. 进入实例设置 → **「概览」**
3. 点击 **「安装 SMAPI」**
4. 选择 SMAPI 版本
5. 等待安装完成

### 安装来源

| 来源 | 说明 |
|------|------|
| **官方** | 从 SMAPI 官方 GitHub 下载 |
| **镜像** | 从国内镜像下载（更快） |
| **本地文件** | 从本地 ZIP 文件安装 |

### 首次安装流程

```
1. 检测游戏版本
      ↓
2. 选择兼容的 SMAPI 版本
      ↓
3. 下载 SMAPI 安装包
      ↓
4. 解压并安装到游戏目录
      ↓
5. 配置启动参数
      ↓
6. 完成！
```

## 版本管理

### 查看已安装版本

在实例概览页面可以看到：

- 当前 SMAPI 版本
- 是否为最新版本
- 支持的游戏版本

### 更新 SMAPI

1. 点击 **「检查更新」**
2. 如果有新版本，点击 **「更新」**
3. 等待更新完成

### 降级 SMAPI

某些 Mod 可能需要特定版本的 SMAPI：

1. 点击 **「版本管理」**
2. 选择需要的版本
3. 点击 **「安装此版本」**

## 启动配置

### 启动参数

SVL 支持自定义 SMAPI 启动参数：

| 参数 | 说明 |
|------|------|
| `--no-terminal` | 禁用终端窗口 |
| `--verbose` | 详细日志输出 |
| `--debug` | 调试模式 |

### 配置方式

1. 进入实例设置 → **「设置」**
2. 找到 **「启动参数」**
3. 添加或修改参数

### 启动模式

| 模式 | 说明 |
|------|------|
| **Mod 模式** | 通过 SMAPI 启动，加载 Mod |
| **原版模式** | 直接启动游戏，不加载 Mod |

## 故障排除

### SMAPI 无法启动

**可能原因：**

- 游戏版本不兼容
- SMAPI 文件损坏
- 权限问题

**解决方法：**

1. 检查游戏版本是否匹配
2. 重新安装 SMAPI
3. 以管理员身份运行

### Mod 加载失败

**可能原因：**

- Mod 版本不兼容
- 缺少依赖 Mod
- Mod 文件损坏

**解决方法：**

1. 检查 Mod 支持的 SMAPI 版本
2. 安装缺失的依赖
3. 重新下载 Mod

### 查看日志

SMAPI 日志位置：
```
%LocalAppData%\SVL\instances\{实例名}\logs\
```
